#### Bidirectional reflectance distribution function

```
Lo(v)=∫ f(l,v)Li(l)⟨n⋅l⟩dl
f(l, v) = fd(l, v) + fr(l, v)
fd(l, v) = cdiff / π
fr(l, v) = D(h) G(l, v) F(l, v) / 4⟨n⋅l⟩⟨n⋅v⟩
```
